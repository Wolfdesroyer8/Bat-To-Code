# Bat-To-Code
## Compile Instructions
First dowload all the files as a zip or use github desktop to clone it.
Next download nodejs [here](https://nodejs.org/en/ "Nodejs").
Then if you downloaded the zip extract it.
Next in the directory you have the files in open cmd and run this command.
```bat
npm install electron
```
Then run this command to run the app.
```bat
npm start
```
## Install Instructions
First go to the projects tab. 
Then go into Bat-To-Code and download the installer. 
Finally just run the installer to download the complete verison of Bat_To-Code.
